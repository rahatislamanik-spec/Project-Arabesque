# Project Arabesque — NBS IT Automation System

Automated IT onboarding and offboarding system built for 
Canada's National Ballet School (NBS) — a boarding school 
serving international students from around the world.

Built entirely on Microsoft 365. No new software. No vendor contracts.

## What's Automated
- International student IT onboarding (Forms → SharePoint → Email → IT Alert)
- Student offboarding (Account disable → License reclaim → Audit log)
- IT Asset Inventory tracking

## Stack
Microsoft Forms · Power Automate · SharePoint Online · 
Office 365 Outlook · Microsoft Teams · M365
